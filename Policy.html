Privacy Policy

Last updated: November 2025

1. Introduction

This Privacy Policy explains how the individual creator operating the "InstaClashed" content channel manages data related to the use of their private software tools ("Service"). As this Service is for internal, individual use, data collection is minimal.

2. Data Collection and Use

The Service is designed to operate **locally** whenever possible. The following data points *may* be collected or processed:

  • Simulation Data: Replay files (.replay) and statistics files (.stats.json) generated during simulation runs are stored locally in the designated replay/data directories.
  • Campaign/Job Data: Data stored in the local database (database.json) contains job titles, player names, dates, and status flags.
  • External API Data: If the Caption Generation feature is used, the input data (job titles, player names) and the response from the third-party AI service (Cerebras) are processed temporarily in memory to generate the title. This data is not intentionally stored long-term by the Service operator, though it passes through external servers.

No personally identifiable information (PII) of third parties (e.g., viewers, customers) is collected or stored by this Service.

3. Data Sharing

No data collected or generated by the Service is shared with any third parties, except as strictly necessary for the operation of external services like the AI API for caption generation.

4. Jurisdiction

The User (creator) is based in **Lithuania**, and the data processing adheres to the context of individual, private creative activity.
